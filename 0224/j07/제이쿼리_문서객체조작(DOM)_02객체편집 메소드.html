제이쿼리 문서객체조작_01객체편집 메소드 p233

객체 편집 메소드는 html문서안에서 사용자가 선택한 요소의 앞쪽이나 뒤쪽에 내용을 추가하거나 복제, 삭제하는 메소드.

1. 내용을 추가하기(add)

  • append(); - 선택한 요소(부모)의 안쪽으로 맨뒤에 내용을 추가하기
  • prepend(); - 선택한 요소(부모)의 안쪽으로 맨앞에 내용을 추가하기
  • after(); - 선택한 요소(형제)의 뒤에 내용을 추가하기
  • before(); - 선택한 요소(형제)의 앞에 내용을 추가하기
  • insertBefore(); insertAfter(); - 자리배치하기

문단요소입니다.

2. 내용을 삭제하기(Remove)

사용자가 선택한 요소를 찾아서 내용만 지우거나, 요소자체를 삭제하기

  • remove(); 사용자가 선택한 요소를 제거
  • empty(); 사용자가 선택한 요소의 내용만 제거
empty버튼을 클릭하면 이곳의 내용이 지워집니다.
remove버튼을 클릭하면 이 박스가 제거됩니다.
