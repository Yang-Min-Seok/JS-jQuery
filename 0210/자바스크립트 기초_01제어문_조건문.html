04자바스크립트 기초_제어문-p.60

자바스크립트 제어문은 프로그램의 흐름을 제어하기 위해서 사용한다. 제어문의 종류에는 조건문, 선택문, 반복문이 있으며, 프로그램 작성시 많이 활용된다.

1. 조건문 - if, else, else if

조건문은 사용자가 조건식을 만들어서 조건에 만족하면 실행이 되고, 조건에 만족하지 않으면 다른 결과가 나오도록 하는 제어문이다.

	작성법)
	조건식 1개일 경우
			if(조건식){
			 조건에 맞는 경우 실행할 내용;
			}
			
	조건식 2개일 경우
			if(조건식){
			 조건에 맞는 경우 실행할 내용;
			}	else {
			 조건에 맞지 않은 경우 실행할 내용;
			}	
			
	조건식이 여러개일 경우
	 if(조건식){
	   조건에 맞을 경우 실행할 내용;
	 }else if(조건식2){
	   조건에 맞을 경우 실행할 내용;
	 }else if(조건식3){
	   조건에 맞을 경우 실행할 내용;
	 }else if(조건식4){
	   조건에 맞을 경우 실행할 내용;
	 }else{
	   그렇지 않은경우 실행할 내용;
	 }
